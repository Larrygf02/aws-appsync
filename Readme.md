#APP SYNC TUTORIAL

### Install amplify cli

```
npm install -g @aws-amplify/cli
```

### Configure amplify

```
amplify configure
```

### Create react-app
```
npx create-react-app <app-name> --typescript
```

### Init amplify in your react project
```
amplify init
```

### App api graphql
```
amplify add api
```

### Push api
```
amplify push
```
